model api

$set

set key/value to model

  • @param {String} key
  • @param {Object} value
  • @return {Object} Error object when set value failed
  • @api public

$get

get data from model

  • @param {String} key
  • @return {Object} data value

$pack

pack data object to model

  • @param {Object} data
  • @return {Object} Error object when pack data failed
  • @api public

$packResultSet

pack database resultSet data to model

  • @param {Object} resultSet data
  • @return {Object} Error object when pack resultSet failed

$before

set before filters to model

filter can be String which is the name of the filter method in the model
or can be Array which contains the filter methods in order

examples:

  • @param {String|Array} before filter
  • @api public

$after

set after filters to model

filter can be String which is the name of the filter method in the model
or can be Array which contains the filter methods in order

  • @param {String|Array} after filter
  • @api public
